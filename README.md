#  Paint Application

A professional, feature-rich paint application built with Python Tkinter and Pillow.

##  Features

###  Drawing Tools
- **Pencil Tool** - Freehand drawing with adjustable width
- **Eraser Tool** - Erase with customizable size
- **Line Tool** - Draw straight lines
- **Shape Tools** - Circle, Rectangle, Triangle, Star, Hexagon, etc.

###  Color Management
- **Color Picker** - Choose any color from palette
- **Quick Colors** - Predefined color palette
- **Canvas Color** - Change background color
- **Fill Tool** - Fill shapes with color

###  Advanced Features
- **Zoom** - Zoom in/out functionality
- **Undo/Redo** - Multiple level undo/redo
- **Save/Load** - Save as JPG/PNG, load images
- **Shape Filling** - Fill existing shapes

###  Shape Library
- Basic: Circle, Rectangle, Triangle, Oval
- Polygons: Pentagon, Hexagon, Heptagon, Octagon, Nanogon, Decagon
- Stars: 5-point star
- Lines: Straight lines

##  Installation

### Prerequisites
- Python 3.8+
- Tkinter (usually comes with Python)
- Pillow library

# Install dependencies
pip install -r requirements.txt

# Run application
python src/Hassan.py
